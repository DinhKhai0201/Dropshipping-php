JS:		https://github.com/mishoo/UglifyJS2

	Install:	npm install uglify-js -g

	Single file:

		Command: 		uglifyjs [input-file] --output [output-file]

		EX: 		uglifyjs jquery-3.2.1.js --output jquery-3.2.1.min.js

	Multi files:
		
		Command:	uglifyjs [input files] [options]


*NOTE: if can not minify (have error), please go to https://javascript-minifier.com/ to compress js then create js min file manual
